Segue as informações do módulo Entrega da Bertholdo Consultoria

Passos para utilização do módulo

1 - Gere o arquivo CSV para poder modificar as informações dos valores clicando no botão -> Gerar arquivo CSV Padrão;
2 - Após o passo anterior efetue o download do arquivo CSV para sua máquina e modifique os valores no arquivo;
3 - Após modificar o arquivo CSV selecione (Selecionar arquivo...) escolha o arquivo CSV modificado;
4 - Click em (Importar);
5 - Verifique as faixas de CEP e valor;

Ao instalar o módulo o mesmo já vem populado com um arquivo CSV padrão, basta gerar o arquivo e modifica-lo.

Ao instalar o módulo limpe as pastas var/cache e var/session e realize um novo login.

O módulo Brt Entrega/Delivery foi idealizado pela empresa Bertholdo Consultoria, com a finalidade de ajudar o administrador da loja virtual no processo de entrega de seus produtos aos clientes utilizando seus próprios meios de entrega tais como motoboy, carro, caminhão e etc.
O processo é bem simples, basta gerar o arquivo CSV e modificar as informações dos ranges de CEP, caso queira cadastrar um novo range através do arquivo basta inserir as informações do mesmo, lembrando sempre de seguir o formato do arquivo e realizar a importação do mesmo via administração. Criamos um video que demonstra a utilização do módulo seguindo todos os passos necessários visando assim facilitar a utilização do mesmo.

Campos do CSV padrão

Cep_Inicial-> Representação da faixa de cep inicial (Ex: 34000000);
Cep_Final -> Representação da faixa de cep final (Ex: 35000000);
Peso_1 -> Representação da faixa de peso em kg inicial (Ex: 0);
Peso_2 -> Representação da faixa de peso em kg final (Ex: 10);
Valor_Compra -> Se o valor da compra for <= (Valor_Compra) aplico frete integral caso contrário aplico frete especial
Valor_Sem_Desconto -> Frete integral (frete normal)
Valor_Com_Desconto -> Frete especial (reduzido)

A pasta media deve estar com permissão 0777.

English

Follow the information delivery module Bertholdo Consulting

Steps to use the module

1 - Generate the CSV file in order to modify the information of the values by clicking the -> Generate CSV Standard;
2 - After the above please download the CSV file to your machine and step modify the values in the file;
3 - After modifying select the CSV file (Select File ...) choose the modified CSV file;
4 - Click on (Import);
5 - Check the CEP and value ranges;

When installing the module it is already populated with a standard CSV file, just create the file and modify it.

When installing the module clean folders var/cache and var/session and perform a new login.

The Brt Delivery module was designed by the company Bertholdo Consulting, in order to help the administrator of the virtual store in the delivery process of their products to customers using its own means of delivery such as a motorcycle courier, car, truck and so on.
The process is very simple, just generate the CSV file and modify the information of the postal code ranges, if you want to register a new range through the file just enter the information yourself, always remembering to follow the format of the file and importing the same route of administration. We created a video that demonstrates the use of the module following all the necessary steps aiming thus facilitate its use.

Fields of the CSV standard

Cep_Inicial-> Representation Band initial cep (Ex: 34000000);
Cep_Final -> Representation Band cep end (Ex: 35000000);
Peso_1 -> Representation of the initial weight range in kg (Ex: 0);
Peso_2 -> Representation of the weight range in late kg (Ex: 10);
Valor_Compra -> If the purchase amount is <= (Valor_Compra) apply full freight otherwise apply special shipping
Valor_Sem_Desconto -> Full Shipping (Standard Shipping)
Valor_Com_Desconto -> shipping special (reduced)

The media folder should be with permission 0777.
